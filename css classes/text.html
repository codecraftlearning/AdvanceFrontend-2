Difference between em, rem, %, and px for text sizing.

px (Pixels) — Absolute Unit

 
Fixed size. Doesn’t depend on anything else.

1px = one screen pixel (roughly).

Doesn’t change if parent size or browser zoom changes (except user zoom).



em — Relative to Parent’s Font Size

1em = current element’s parent font size.

So, if the parent is 20px and you set font-size: 2em, it becomes 40px.



rem — Relative to Root (HTML) Font Size

1rem = root element (< html >) font size, not parent.

Stays consistent everywhere.

If html { font-size: 10px; }, then 1rem = 10px.



% (Percentage) — Relative to Parent Font Size

Works like em, but written as a percent.

100% = same as parent’s size.

150% = 1.5 × parent’s size



PX vs EM vs REM vs %

Unit Relative To Example Changes with Parent? Common Use
px Nothing (fixed) 16px No Fixed-size elements
em Parent’s font size 1.5em Yes Buttons, nested text
rem Root font size 1.5rem No Global scaling, accessibility
% Parent’s font size 120% Yes Inline adjustments



Cover vs Contain

Property What It Does Crops Image? Fills Entire Area? Leaves Blank Space? Common Use
cover Scales image to cover entire container Yes Yes No Backgrounds, banners
contain Scales image to fit inside container No No Yes Logos, icons, full view



Text Overflow and Wrapping

Example CSS Properties Result
Normal Wrap white-space: normal;
overflow: visible;
text-overflow: none;
This is a very long text that will wrap onto multiple lines and no ellipsis will appear.
No Wrap, Overflow Visible white-space: nowrap;
overflow: visible;
text-overflow: none;
This is a very long text that will overflow outside the box but no ellipsis will appear.
No Wrap, Ellipsis white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
This is a very long text that will be truncated with an ellipsis if it does not fit.
